SQL Functions: It is a built-in SQL code:
					- Accepts an input value
					- Processes it
					- Returns an output value

Mainly we have 2 types of functions: 1. Single row functions
									 2. Multi row functions
In single row functions we have: String functions
								 Numeric functions
								 Date & Time functions
								 Null functions
In multi row functions we have: Aggregate functions (Basic)
								Window functions (Advanced)
